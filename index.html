ABOUT THIS SITE

Welcome to our website!

This project was created by Vinicius and completed on 07/25/2024. Initially developed as a project for the Federal Technological University of Paraná (UTFPR), the website is inspired by the design and functionality of a local movie theater.

This was my first major project and an enriching experience. I learned a lot throughout the process and used several technologies, such as JavaScript, Bootstrap, and an API for requesting movies.

It is important to note that the information presented on the website does not represent reality; it was used only for academic purposes. The development of this website required a lot of work and dedication, and I am very pleased with the results obtained.

In addition, I made an effort to make the website as configurable as possible. The times and the number of movies available each day are changed, in addition to the times being dynamic, adjusting automatically based on the current time and day.
